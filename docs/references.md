# References

## Technologies Used

### Backend
- [FastAPI](https://fastapi.tiangolo.com/) — Modern Python web framework
- [Motor](https://motor.readthedocs.io/) — Async MongoDB driver for Python
- [Pydantic](https://docs.pydantic.dev/) — Data validation using Python type hints
- [pydantic-settings](https://docs.pydantic.dev/latest/concepts/pydantic_settings/) — Settings management
- [Uvicorn](https://www.uvicorn.org/) — ASGI server for FastAPI
- [psutil](https://psutil.readthedocs.io/) — System metrics (CPU, RAM)
- [pynvml](https://pypi.org/project/pynvml/) — NVIDIA GPU metrics
- [OpenCV Python](https://opencv.org/) — RTSP capture and frame processing

### Infrastructure
- [MediaMTX](https://github.com/bluenviron/mediamtx) — RTSP/RTMP/HLS media server
- [FFmpeg](https://ffmpeg.org/) — Video processing and RTSP publishing
- [MongoDB](https://www.mongodb.com/) — Document database
- [Docker](https://www.docker.com/) — Containerization

### Frontend
- [Lucide Icons](https://lucide.dev/) — Icon library (same as in TSX reference)
- [Google Fonts — Inter](https://fonts.google.com/specimen/Inter) — UI typography
- [Google Fonts — JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) — Monospace for metrics

## Standards and Patterns
- [MJPEG multipart/x-mixed-replace](https://en.wikipedia.org/wiki/Motion_JPEG) — Browser video streaming
- [RTSP RFC 2326](https://tools.ietf.org/html/rfc2326) — Real Time Streaming Protocol
- [WebSocket RFC 6455](https://tools.ietf.org/html/rfc6455) — WebSocket protocol
- [Python asyncio docs](https://docs.python.org/3/library/asyncio.html) — Async/await patterns
